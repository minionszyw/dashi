# 国学大师 - AI命理分析小程序

## 📋 项目简介

基于AI的命理分析微信小程序，为用户提供智能化的命理咨询服务。

### 核心特性
- 🤖 AI驱动的智能对话
- 💬 多轮会话管理
- 💰 Token计费系统
- 🔐 微信授权登录
- ⚡ 流式响应优化体验

## 🏗️ 技术栈

### 前端
- **框架**: uni-app
- **UI**: uni-ui
- **语言**: TypeScript + Vue 3

### 后端（计划）
- **框架**: FastAPI
- **数据库**: PostgreSQL
- **缓存**: Redis
- **AI**: LangChain + DeepSeek
